# AI Career Gap Analyzer

Bridging the gap to your dream career with AI-powered insights.

## Overview

The **AI Career Gap Analyzer** helps students and professionals identify skill gaps between their current resume and their dream job. Using advanced AI models, it analyzes your profile and generates a personalized, week-by-week learning roadmap with curated free resources.

## Setup & Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/varshatech-17/Carrer-Path-Roadmap.git
    cd Carrer-Path-Roadmap
    ```

2.  **Create a virtual environment:**
    ```bash
    python -m venv venv
    source venv/bin/activate  # on Windows: venv\Scripts\activate
    ```

3.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Configure Environment Variables:**
    -   Rename `.env.example` to `.env`
    -   Add your `GROQ_API_KEY` (Get one from [console.groq.com](https://console.groq.com/))

5.  **Run the application:**
    ```bash
    python main.py
    ```
    Visit `http://127.0.0.1:8000` in your browser.
